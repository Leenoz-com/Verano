# Title
Verano Theme

## Description
Verano is a sleek and modern phpBB ACP template, fully responsive and designed for effortless navigation and ease of use.

## Features
- Modern Theme
- Easy to Install
- Easy to Customize
- W3C Valid Code
- Fully Responsive
- RTL Support
